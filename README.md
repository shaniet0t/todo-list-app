A feature-rich, responsive To-Do List application built with HTML, CSS, and JavaScript.  
Easily manage daily tasks with editing, priorities, due dates, and more. Tasks persist via local storage—ideal for personal, work, or school use.

## Features

- **Add, Edit, Delete Tasks:** Create tasks, edit their details, or remove them.
- **Task Completion:** Mark tasks as complete/incomplete with a single click.
- **Due Dates:** Optionally set a due date for tasks.
- **Task Priorities:** Set priority levels (Low, Medium, High) and sort/filter your list.
- **Categories/Lists:** Organize tasks under custom categories (e.g., Work, Personal).
- **Search & Filter:** Find tasks by text, category, completion, priority, or date.
- **Bulk Actions:** Complete or delete multiple tasks at once.
- **Progress Indicator:** Visual progress bar shows how much you’ve accomplished.
- **Notes & Comments:** Add notes to tasks for details.
- **Responsive Design:** Works on desktop, tablet, and mobile.
- **Accessible:** Keyboard navigation and ARIA labels for screen readers.
- **Local Storage Persistence:** All tasks saved in your browser—even if you close it.
- **Data Export/Import:** Export tasks to CSV/JSON, import tasks from a file.
- **Customizable Themes:** Choose between light/dark mode and more.
- **Activity Log:** See a history of your actions (task added, edited, completed).

## How to Use

1. Open `index.html` in your browser.
2. Type a new task, set priority/due date/category (optional), and click **Add**.
3. Click a task to mark completed/uncompleted. Click **✎** to edit, **✕** to delete.
4. Use filters and search to organize your tasks.
